# IA treinada pela dataset CIFAR10

# Descrição
O projeto consiste em uma criação de uma
IA convolucionária que e treinada pelo dataset
CIFAR10 e depois há um teste com o restante 
do dataset.

# Tecnologias Usadas
- Python
- Pytorch
  
# Como Executar
Salve os arquivos main.py e redes.py
na mesma pasta, depois compile o
arquivo main.
